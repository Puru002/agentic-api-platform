# Agentic API Platform

## What is this?
A platform to orchestrate AI agents using APIs and services.

## Architecture
Agents → APIs → Services

## Components
- agents/
- apis/
- services/
- docs/

## How it works
1. User sends request
2. Agent interprets intent
3. API routes request
4. Service executes
5. Response returned

## Docs
See /docs

## Run
cd services/api-service
npm install
node index.js
